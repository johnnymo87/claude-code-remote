# CLAUDE.md - Claude Code Remote

Control Claude Code remotely via Telegram, Email, LINE, or Desktop notifications.

## Quick Start

```bash
npm install
npm run setup    # Configure channels, set up hooks
npm run webhooks # Start notification services
```

## Commands

| Command | Description |
|---------|-------------|
| `/test-notify` | Test all enabled notification channels |

## Skills

| Skill | When to use |
|-------|-------------|
| `architecture` | Understanding how it works, project structure |
| `configuring-notifications` | Setting up Email, Telegram, or LINE channels |
| `troubleshooting` | Notifications not working, commands not executing |
